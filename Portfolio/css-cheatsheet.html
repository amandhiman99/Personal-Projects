Selectors
Selectors are patterns used to select the elements we want to style. Some of them are
*
It selects all the elements present in the HTML
div
It selects all the div tag present in the HTML to style
div,p
It selects all the div tags and paragraph tags to style
div p
It selects all the paragraph tags inside the HTML to style
div + p
It selects all the p tags immediately after the div tags to style.
CSS -Properties
It is the list of properites used in the CSS to style the content. Some of them are
align-content
It is the behavior of the flex-wrap property
align-items
It is for the alignment of items inside the container
animation
It binds an animation to an element
animation-delay
It delays the start of the animation
background
It defines all the background properties as once.
