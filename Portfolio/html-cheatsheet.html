HTML Elements
The HTML elements consist of a starting tag and if the elements contain some content then it ends with a closing tag. There are many type of HTML elements some of them are listed
Block Elements:
<div> It creates a new division in the webpage </div>
<p> It makes a new paragraph </p>
<h1> It creates a heading in the webpage </h1>
<ul> It creates an unordered list </ul>
<ol> It creates a ordered list <ol>
Inline-Elements:
<em> It emphasise on the text </em>
<strong> It bolds the text </strong>
<sup> It defines the superscript text.</sup>
<sub> It defines the subscript text. </sub>
HTML Attributes
The HTML Attributes give a desired element additional information to either modify the elements functionality or provide functionality. Some them are listed
href Attribute
It adds a link into your webpage.
<a href = "type the link here " >
src Attribute
This attribute adds an image to the webpage.
<img src = "image name or link" >
The lang Attribute
It Determines the language of the document. It is done in the html tag
<htmllang ="any language" >
The style Attribute
The style attibute is used to specify the styling of the element.
<p style = "any attributes here" /p>

Input Element
The <input> Element
The input element is the most important form element. It can be displayed in several types.
Type Description
<input type = "text" > It defines one line text input field
<input type = "radio"> It defines a button to select from many choices
<input type= "submit"> It defines a submit button
